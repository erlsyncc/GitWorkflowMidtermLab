# Git Workflow Midterm Lab

This repository contains a simple starter web project with a static HTML page, CSS styling, and JavaScript behavior.

## Files

- `index.html` – main HTML page
- `style.css` – page styling
- `script.js` – JavaScript behavior
- `.gitignore` – Git ignore rules

## Usage

1. Open `index.html` in a browser.
2. Edit the source files as needed.
